# Air SDK Documentation

This directory contains documentation about specific functionality of the Air SDK. The documentation is provided in the form of Jupyter notebooks that demonstrate how to use various features of the SDK with code examples.

## Available Documentation

- [Basic Simulation CRUD](./examples/simulations.ipynb) - Basic CRUD operations on simulations.
- [Import and Export](./examples/import_export.ipynb) - Documentation on importing simulations from export files or data, and exporting existing simulations.
- [Out-of-band Networks (OOB)](./examples/oob.ipynb) - Adding/removing automatic OOB networks
- [Starting and Stopping Simulations](./examples/starting_and_stopping_simulations.ipynb) - Running simulations.
- [Simulation History](./examples/history.ipynb) - Reading the history of a simulation.
- [ZTP Scripts](./examples/ztp_scripts.ipynb) - CRUD operations on ZTP scripts of a simulation
- [Images](./examples/images.ipynb) - Create, update, upload, download, and delete images
- [Nodes](./examples/nodes.ipynb) - Basic CRUD operations on nodes
- [Workers, Fleets and Certificates](./examples/workers_fleets_and_certificates.ipynb) - CURD and management of worker machines

## Files Directory

The `files/` directory contains example files used in the documentation notebooks:

- `files/ztp/` - Example ZTP script files
- `files/exports/` - Example simulation export files
